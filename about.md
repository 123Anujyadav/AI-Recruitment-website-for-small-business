# TalentAI - Smart AI-Powered Job Matching Platform

**TalentAI** is an intelligent recruitment platform that connects the right talent with the right opportunities using advanced AI algorithms. By combining semantic skill matching with location-based intelligence, we revolutionized how employers find candidates and how job seekers land their dream jobs.

---

## Why TalentAI Stands Out

### 1. Intelligent Match Scoring (0-100)
Our proprietary algorithm evaluates every candidate-job pair with a precise **Match Score** calculated from:
- **Skill Synergy (70%)**: Deep semantic analysis of technical and soft skill overlap
- **Geographic Optimization (30%)**: Location-aware ranking that prioritizes local talent for better retention

### 2. Pincode-Aware Fair Ranking
We ensure fair hiring by guaranteeing that local candidates always rank above out-of-area applicants when skills are comparable—reducing relocation friction and improving hiring success rates.

### 3. AI-Powered Career Intelligence
Integrated with **Google Gemini 2.5 Flash** to provide:
- **Real-time Salary Assessment**: Market-standard salary analysis for any role
- **Personalized Learning Paths**: Custom-generated skill-gap roadmaps with resources and timelines

### 4. Complete Recruitment Ecosystem

**For Candidates:**
- One-click job applications
- Real-time application tracking (Applied → Shortlisted → Rejected)
- AI-generated career guidance
- Walk-in job support with physical interview locations

**For Employers:**
- Automated candidate ranking by Match Score
- Applicant management dashboard
- Shortlist/reject workflow
- Detailed matching metrics

## API Architecture

TalentAI is built as a modern REST API with:
- Complete CRUD operations for jobs, applications, profiles
- Role-based access control (Candidate / Employer / Admin)
- Real-time match score calculation
- Admin dashboard for platform management
