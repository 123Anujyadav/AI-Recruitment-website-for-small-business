# Product Requirement Document (PRD)

## TalentAI -- MVP Job Matching & Candidate Ranking System

## 1. Objective

Build a simple recruitment platform for small companies focusing only
on: - Candidate Registration & Profile - Job Posting - Skill-Based
Matching - Candidate Ranking

No advanced AI, embeddings, or salary prediction in MVP.

## 2. Target Users

-   Small Employers / HR
-   Local Job Seekers

## 3. Core Features

### Candidate

-   Register/Login
-   Create Profile (Skills, Experience, Pincode)
-   View Matched Jobs
-   Apply to Jobs

### Employer

-   Register/Login
-   Post Job (Title, Required Skills, Pincode)
-   View Ranked Candidates
-   Update Application Status

## 4. Success Metrics

-   Time to shortlist candidate \< 5 minutes
-   Accurate skill-based ranking
-   Hyper-local matching via pincode
